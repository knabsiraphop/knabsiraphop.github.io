A set of reusable Unity tooling published as open-source UPM packages under the **KidzDev** brand — extracted from patterns I rely on across production titles.

## What's in it

- **Addressables toolkit** — ref-counted loading, pooling, and typed asset references for safer, leak-free Addressables usage.
- **Recyclable uGUI ScrollRect** — a cell-recycling scroll view for long lists without a GameObject per item.
- **Responsive UI sizing** — utilities for adapting UI across aspect ratios and resolutions.
- **C# extensions** — small quality-of-life helpers used throughout my projects.

All available on my GitHub.
