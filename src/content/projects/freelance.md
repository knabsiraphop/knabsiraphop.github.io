Three Unity projects delivered for clients alongside studio work.

## Mascot Design

A standalone Unity **desktop app** (Spine 4.0) for creating and customizing a 2D mascot, then exporting a high-resolution PNG / JPG. Stack-based panel navigation, a Spine skin customizer (attachment remapping + per-part color tinting), bone scaling, custom image import via native file dialogs, JSON save / load, and offscreen RenderTexture export.

## SolarYield

A single-scene mobile game prototype simulating a solar-panel yield farm — a compressed day / night cycle, electricity yield, random weather events, an in-game economy (SOLA coin), and a marketplace. Built on a message-bus architecture with abstract system / UI base classes, skybox cycling, and dynamic lighting.

## AvatarCustom

A runtime avatar-customization system for a Spine 2D character — non-destructive skin-swapping via attachment remapping, per-slot skin-tone and hair-color tinting (3 tones, 18 hair colors), and a data-driven part → slot mapping, with custom editor inspectors.
